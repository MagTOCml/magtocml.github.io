This is a Work-In-Process Project

As originally envisioned, the infrastructure of this project was to be developed around a Twitter-bot workflow. However, for some unclear reason my application for Twitter developer status was denied. Rather than appeal that decision, I have shifted to a workflow based on a freely available TOCspotting app and voluntary contributions to a GitHub TOCdata repository.

The current plan is to provide the Python-based tool to anyone with the skills and interest to download and install the tool script. The next step will be the revision of the tool to use the Beeware UI/UX framework rather than its current dependency on wxPython wxWidgets framework. This rewrite will allow the production of multi-platform enduser-installable apps that will broaden the community that can help with the creation of the MagTOC dataset.

For now, please find the tool, dataset, and this site's repositories on the MagTOCml organization page at GitHub.
